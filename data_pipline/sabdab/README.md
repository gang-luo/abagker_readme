# SabDab Data Preparation

## Source
Download the SabDab archive and metadata from the official portal:
- https://opig.stats.ox.ac.uk/webapps/sabdab-sabpred/sabdab

Store the downloaded files in this directory before running parsing notebooks.

## Processing Goal
The SabDab stage extracts antibody/antigen chain sequences and metadata required for AbAgKer sample construction, then exports normalized JSON entries for later merging with other sources.
